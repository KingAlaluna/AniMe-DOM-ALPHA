# AniMe-DOM
Сайт для перегляду аніме, підключений на AniLibria API. Проєкт не закінчений та знаходиться в активній розробці. (Я працюю з AniLibria API (та всіми **іншими API AniLibria** (тому що для багатьох дій (наприклад показати зображеня) потрібно **окреме API**)) вперше, і вивчаю його прямо під час розробки цього сайта, тому: 
- 1: розробка займе більше часу.
- 2: в alpha версіях буде багато дублюючого коду для кращого ознайомлення та **вивчення api**, це зроблено **свідомо** зваживши всі плюси та мінуси. В фінальній версії код буде **повністю** переписаний під максимальну оптимізацію, підтримуваність, та **видаленням всіх дублів**.
